## 1. 서비스 정의

본 서비스는 **20~30대 여성을 대상으로 한 모바일 웹 기반 AI 사주 서비스**다.

사용자는 자신의 사주 정보를 입력하고, 오늘의 운세, 기본 사주풀이, 연애·관계·궁합·재회·덕질 등 주제별 심화 사주 리포트를 확인할 수 있다.

서비스의 핵심은 단순 챗봇이 아니라, **사주풀이 콘텐츠를 주제별 리포트 형태로 제공하고, 필요 시 AI 챗봇으로 추가 질문을 이어가는 구조**다.

---

## 2. 서비스 포지셔닝

본 서비스는 ChatGPT처럼 사용자가 자유롭게 질문하는 범용 AI 상담 서비스가 아니다.

핵심 포지션은 다음과 같다.

> **가볍게 오늘의 운세를 보고,내 사주 기본 풀이를 확인하고,특정 고민은 주제별 심화 리포트로 깊게 보는 AI 사주 서비스**
> 

챗봇은 서비스의 메인 상품이 아니라, 사주풀이와 리포트를 본 뒤 추가 질문을 처리하는 보조 기능이다.

# 4. 핵심 기능 구조

본 서비스의 핵심 기능은 3개로 구분한다.

1. **오늘의 운세**
2. **부분 블러 처리된 정통사주형 사주풀이**
3. **잼컨 주제별 심화 사주 리포트**

이 3개는 하나의 순차 퍼널이 아니라, **각각 독립된 기능/상품군**이다.

다만 사용 흐름 안에서 서로 연결될 수 있다.

---

# 5. 기능 1. 오늘의 운세

## 5-1. 기능 성격

오늘의 운세는 가장 가벼운 진입 기능이다.

사용자가 매일 부담 없이 확인할 수 있는 무료 콘텐츠이며, 서비스의 습관성과 재방문을 만드는 역할을 한다.

## 5-2. 핵심 목적

- 첫 방문자의 진입 장벽 완화
- 매일 재방문 유도
- 가벼운 재미 제공
- 사주 콘텐츠에 대한 반응 확인
- 심화 리포트 또는 기본 사주풀이로 자연스러운 이동 유도

## 5-3. 제공 범위

오늘의 운세에서는 짧고 즉시 이해 가능한 결과를 제공한다.

제공 항목 예시:

- 오늘의 연애운
- 오늘의 연락운
- 오늘의 한 줄 조언
- 오늘 피해야 할 행동
- 오늘의 행운 키워드
- 오늘의 행동 가이드

## 5-4. 무료/유료 정책

오늘의 운세는 기본적으로 무료 콘텐츠로 운영한다.

단, 오늘의 운세 결과 하단에서 다음 유료 콘텐츠로 연결할 수 있다.

- 오늘의 연락운 상세 리포트
- 상대 마음 흐름 리포트
- 궁합 상세 리포트
- 재회운 리포트
- 기본 사주 전체 풀이

## 5-5. 기획상 역할

오늘의 운세는 직접 매출을 만드는 기능이 아니라, **유입·재방문·전환의 출발점**이다.

따라서 오늘의 운세에서 너무 많은 정보를 제공하지 않는다.

무료 결과는 짧고 완결되게 제공하되, 심화 분석은 별도 유료 상품으로 분리한다.

# 6. 기능 2. 부분 블러 처리된 정통사주형 사주풀이

## 6-1. 기능 성격

이 기능은 사용자의 기본 사주 정보를 바탕으로 제공되는 **기본 사주풀이 콘텐츠**다.

오늘의 운세보다 더 진지하고 정통사주에 가까운 성격을 가지며, 사용자가 “내 사주를 제대로 봤다”고 느끼게 하는 역할을 한다.

## 6-2. 핵심 목적

- 사주 서비스로서의 신뢰 형성
- 사용자의 기본 성향 이해 제공
- 첫 유료 결제 유도
- 이후 주제별 심화 리포트 구매로 연결
- 챗봇 추가 질문의 기반 제공

## 6-3. 제공 구조

이 기능은 **일부 무료 공개 + 일부 블러/잠금 처리** 구조로 운영한다.

무료 공개 영역에서는 사용자가 결과를 받았다고 느낄 수 있을 만큼의 핵심 요약을 제공한다.

블러 처리 영역에서는 상세 분석 항목을 잠가 유료 해금으로 연결한다.

## 6-4. 무료 공개 영역

무료로 제공 가능한 항목:

- 기본 사주 성향 요약
- 핵심 키워드
- 연애 성향 일부
- 관계에서의 기본 태도
- 현재 흐름 일부
- 오늘의 간단 조언

무료 영역은 반드시 짧게라도 완결되어야 한다.

즉, 사용자가 “결론을 보려면 결제해야 한다”고 느끼면 안 된다.

## 6-5. 블러/잠금 영역

블러 처리 가능한 항목(핵심 조언은 피하기):

- 상세 성향 분석
- 관계에서 반복되는 약점
- 연애 패턴 상세
- 나에게 맞는 상대 유형
- 올해 연애 흐름
- 중요한 시기
- 궁합 가능성
- 갈등 포인트
- 심화 조언

## 6-6. 결제 구조

이 기능의 과금은 **전체 사주풀이 해금** 방식이 적합하다.

예상 상품:

- 기본 사주 전체 풀이 해금
- 연애 성향 전체 해금
- 올해 흐름 포함 전체 해금

예상 가격대:

- 2,900원~3,900원

## 6-7. 기획상 역할

정통사주형 사주풀이는 서비스의 **신뢰 형성 기능**이다.

오늘의 운세가 가벼운 진입이라면, 이 기능은 사용자가 서비스에 대해 “내 정보를 바탕으로 해석해준다”고 느끼게 만드는 역할을 한다.

또한 주제별 심화 리포트 구매 전, 사용자가 자신의 사주 결과를 미리 경험하는 중간 단계로 작동한다.

# 7. 기능 3. 주제별 심화 사주 리포트

## 7-1. 기능 성격

주제별 심화 사주 리포트는 본 서비스의 **핵심 유료 상품**이다.

사용자는 특정 고민이나 관심 주제를 선택하고, 해당 주제에 대한 심화 사주풀이 리포트를 구매한다.

## 7-2. 핵심 목적

- 핵심 매출 창출
- 사용자의 구체적 고민 해결
- 연애·관계·궁합 등 고관여 주제 대응
- 리포트 기반 추가 질문 결제 유도
- 이메일 저장 또는 재열람 가치 제공

## 7-3. 주제별 리포트 카테고리

초기 MVP에서는 연애·관계 중심 리포트를 우선 제공한다.

### 1차 핵심 리포트

- 오늘의 연락운 상세 리포트
- 상대 마음 흐름 리포트
- 궁합 상세 리포트
- 재회운 리포트
- 내 연애 패턴 리포트
- 올해 애정운 리포트

### 2차 확장 리포트

- 최애와 나의 가상 케미 리포트
- 최애 활동운 리포트
- 팬심 궁합 리포트
- 금전운 리포트
- 커리어운 리포트
- 다이어트 운세 리포트

## 7-4. 초기 우선순위

초기 MVP에서는 다음 순서로 우선순위를 둔다.

1. 오늘의 연락운 상세
2. 상대 마음 흐름
3. 궁합 상세
4. 재회운
5. 내 연애 패턴
6. 올해 애정운
7. 최애 가상 케미

초기 매출 검증은 연애·관계·궁합 중심으로 진행한다.

최애 콘텐츠는 결제보다 공유와 유입 확산에 더 적합한 보조 카테고리로 본다.

## 7-5. 리포트 구성

유료 심화 리포트는 주제별로 다르지만, 기본적으로 다음 구조를 따른다.

- 현재 흐름 요약
- 사주 기반 해석
- 점수 또는 유형
- 주제별 상세 분석
- 주의해야 할 행동
- 추천 행동 가이드
- 추천 문구 또는 실천 팁
- 일정 기간 흐름
- 한 줄 결론
- 추가 질문 연결

초기 가격은 낮은 진입 가격과 핵심 리포트 가격을 구분한다.

## 7-7. 기획상 역할

주제별 심화 리포트는 서비스의 매출 중심이다.

사용자는 “질문 1회”보다 “완성된 리포트”에 더 쉽게 돈을 낼 가능성이 높다.

따라서 서비스의 유료 전환은 챗봇 질문권보다 **주제별 리포트 구매**를 중심으로 설계한다.

# 8. 챗봇 기능

## 8-1. 기능 성격

챗봇은 독립적인 메인 기능이 아니라, 오늘의 운세, 정통사주형 사주풀이, 주제별 심화 리포트 이후에 붙는 **추가 상담 기능**이다.

즉, 챗봇은 서비스의 중심이 아니라 결과 콘텐츠를 보완하는 기능이다.

## 8-2. 핵심 목적

- 사용자의 추가 질문 대응
- 리포트 해석 보완
- 상황별 행동 가이드 제공
- 추가 질문권 결제 유도
- 다른 리포트 추천
- 사용자 체류 시간 증가

## 8-3. 진입 방식

챗봇은 빈 입력창으로 시작하지 않는다.

먼저 예상 질문 리스트를 제공하고, 사용자가 선택하거나 직접 질문하도록 한다.

예상 질문 예시:

- 오늘 먼저 연락해도 될까?
- 어떤 문장으로 보내면 좋을까?
- 상대가 부담스러워할까?
- 이 관계 계속 가도 될까?
- 이번 주 흐름은 어떻게 바뀔까?
- 내 사주에서 조심해야 할 점은?

## 8-4. ChatGPT와의 차별점

챗봇이 ChatGPT처럼 보이지 않도록 다음 원칙을 둔다.

- 리포트 또는 운세 결과 이후에 연결한다.
- 추천 질문 리스트를 먼저 제공한다.
- 답변은 사주풀이 구조를 따른다.
- 답변은 현재 흐름, 해석, 행동 가이드, 주의점, 한 줄 결론으로 구성한다.
- 범용 상담이 아니라 사주 기반 상담으로 포지셔닝한다.

## 8-5. 무료 질문권

챗봇은 **1일 3개 무료 질문권**을 제공하는 방향을 검토한다.

단, 무료 질문권은 짧은 데일리 질문용으로 제한한다.

무료 질문 가능 범위:

- 오늘 먼저 연락해도 될까?
- 오늘의 한 줄 조언
- 지금 감정 흐름 보기
- 오늘 피해야 할 말
- 간단한 행동 가이드

무료 질문 제한 범위:

- 상대 마음 상세
- 궁합 상세
- 재회 가능성
- 7일 관계 흐름
- 추천 메시지 3개
- 월간/연간 운세
- 정통 사주 상세 풀이

## 8-6. 유료 질문권

심화 질문은 유료 질문권으로 운영한다.

추천 가격:

| 상품 | 추천 가격 |
| --- | --- |
| 심화 질문 1회 | 900원 |
| 심화 질문 3회권 | 2,400원 |

단, 유료 질문권은 핵심 매출 상품이 아니라 보조 매출 상품으로 본다.

주요 매출은 주제별 심화 리포트에서 발생시키는 것이 적합하다.

# 9. 결제 구조

## 9-1. 결제 상품 구분

서비스의 결제 상품은 크게 3개로 구분한다.

1. 정통사주형 사주풀이 전체 해금
2. 주제별 심화 사주 리포트
3. 챗봇 심화 질문권

## 9-2. 무료 콘텐츠

무료 콘텐츠는 다음으로 제한한다.

- 오늘의 운세
- 오늘의 연락운/연애운 한 줄
- 기본 사주풀이 일부
- 오늘 무료 질문 3개
- 공유 가능한 간단 결과

## 9-3. 유료 콘텐츠

유료 콘텐츠는 다음으로 구성한다.

- 정통사주형 기본 풀이 전체 해금
- 주제별 심화 리포트
- 상대 마음 흐름
- 궁합 상세
- 재회 가능성
- 추천 연락 문구
- 7일/30일 흐름
- 심화 질문권

## 9-4. 결제 우선순위

매출 관점에서 결제 우선순위는 다음과 같다.

1. 주제별 심화 사주 리포트
2. 정통사주형 사주풀이 전체 해금
3. 챗봇 심화 질문권

# 13. 주요 리스크 및 대응

## 13-1. 챗봇이 ChatGPT처럼 보일 위험

대응:

- 챗봇을 첫 화면 메인으로 두지 않는다.
- 리포트나 운세 결과 이후 연결한다.
- 예상 질문 리스트를 먼저 제공한다.
- 사주 기반 답변 구조를 유지한다.

## 13-2. 블러 처리가 낚시처럼 보일 위험

대응:

- 핵심 결론은 무료로 제공한다.
- 상세 분석만 블러 처리한다.
- “나머지 보기”보다 “심화 분석 보기”로 표현한다.

## 13-3. 무료 질문 3개가 유료 전환을 낮출 위험

대응:

- 무료 질문은 짧은 데일리 질문으로 제한한다.
- 심화 질문은 유료 질문권 또는 리포트 구매로 연결한다.
- 무료 질문에서 상세 궁합, 재회 가능성, 상대 마음 분석은 제공하지 않는다.

## 13-4. 서비스 범위가 넓어질 위험

대응:

- 초기 MVP는 연애·관계 중심으로 제한한다.
- 최애 콘텐츠는 바이럴 보조 기능으로 둔다.
- 금전운, 커리어운, 다이어트 운세는 후순위 확장으로 둔다.

# 14. 최종 방향

본 서비스의 핵심은 챗봇이 아니라 **사주풀이 콘텐츠의 상품화**다.

따라서 서비스는 다음 3개 기능을 중심으로 구성한다.

1. **오늘의 운세**
    - 무료, 가벼운 데일리 콘텐츠
    - 재방문과 진입 담당
2. **부분 블러 처리된 정통사주형 사주풀이**
    - 일부 무료, 일부 유료 해금
    - 사주 서비스 신뢰 형성 및 첫 결제 유도
3. **잼컴 주제별 심화 사주 리포트**
    - 핵심 유료 상품
    - 연애·관계·궁합·재회 중심 매출 담당

챗봇은 이 세 기능 이후에 붙는 보조 상담 기능으로 운영한다.
